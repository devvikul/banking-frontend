# Beneficiary Management System

This project is a Beneficiary Management System built using React, Redux Toolkit, JSON Server, and other technologies. It allows users to perform CRUD (Create, Read, Update, Delete) operations on beneficiary data. The project includes a form for updating and deleting beneficiaries, a table for displaying beneficiary data, and a view component for viewing individual beneficiary details.

Clone the Project:

Navigate to the project directory:
banking (cd command can be used, if not in project folder)

Install dependencies:
npm install

Start JSON Server
Command for starting json server
json-server --watch db.json --port 5000

Start the development server:

npm run dev
